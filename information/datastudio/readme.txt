Data Studio 5.1.0
========================================================================


ABOUT Data Studio
========================================================================
Data Studio is a universal Integrated Development Environment for developers and database administrators. It simplifies the development and management of OpenGauss Database in both traditional and Cloud deployments. Data Studio is built on the Eclipse?Rich Client Platform (RCP), taking advantage of the RCP framework for building and deploying native GUI applications. For more information on Data Studio, refer to the Data Studio User Manual.

INSTALLATION
========================================================================
1.  Download Data Studio release package   
2.  Unzip the contents to a new folder      
3.  Open the folder. Locate and execute "Data Studio.exe".
